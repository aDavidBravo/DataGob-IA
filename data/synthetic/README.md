# Datos Sintéticos de Desarrollo

Este directorio contiene datos 100% sintéticos generados para desarrollo y pruebas.

> ⚠️ **IMPORTANTE**: Estos datos son ficticios. No representan personas reales.

## Archivos generados por `data_generator.py`

| Archivo | Descripción | Registros (demo) |
|---------|-------------|---|
| `poblacion_segip.csv` | Población sintética Bolivia (SEGIP) | 100,000 |
| `bonos_sociales.csv` | Beneficiarios de bonos (AFP/Municipios) | 15,000 |
| `titulos_universitarios.csv` | Títulos profesionales (MINEDU) | 10,000 |
| `defunciones_sereci.csv` | Certificados de defunción (SERECI) | 8,000 |

## Ejecutar generador

```bash
python src/utils/data_generator.py
```
